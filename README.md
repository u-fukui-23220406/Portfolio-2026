## 1. Java Slot Machine（スロットゲーム）
授業の自由課題として制作した、演出と音にこだわったデスクトップアプリです。

### 🎥 動作デモ

https://github.com/user-attachments/assets/4d59d0c1-00a7-4af8-85ee-1f79568a6932

### 🛠 使用技術
- **言語:** Java 17 (Swing / AWT)
- **AI活用:** Geminiとの対話によるロジック構築（音声再生クラス・重み付け抽選）

### ✨ 開発のこだわりとプロセス
- **AIとの協調開発:** 実装にあたり、生成AI（Gemini）を副操縦士として活用。特に「音声再生のバッファ管理」や「シンボルの重み付け確率計算」において、AIにコードの意図を確認しながら、保守性の高いクラス設計を実現しました。
- **ユーザー体験:** `select.wav`（選択）、`spin.wav`（回転）、`coin.wav`（的中）の3種類の音源を、ゲームの進行状況に合わせて遅延なく再生させることに注力しました。
