## 1. Java Slot Machine（スロットゲーム）

授業の自由課題として制作した、演出と音にこだわったスロットマシーンです。

### 🎥 動作デモ

https://github.com/user-attachments/assets/8173a8c2-3bcc-4278-9c94-0d075fd18d29

### 🛠 使用技術

- **言語:** Java 17 (Swing / AWT)
- **AI活用:** Geminiとの対話によるロジック構築（音声再生クラス・重み付け抽選）

### ✨ 開発のこだわりとプロセス

- **AIとの協調開発（AI Native Development）:** 実装にあたり、生成AI（Gemini）を「サポーター」として活用。特に「音声再生の制御」や「シンボルの重み付け確率計算」において、AIにコードの意図を確認しながら、単に動くだけでなく保守性の高いクラス設計を実現しました。

- **アルゴリズムによるリアリティの追求:**
  - **リール減速の実装:** 単に停止させるのではなく、停止指示後に徐々に速度を落として止まる「減速ロジック」を構築し、実機に近い滑らかな挙動を再現しました。
  - **シンボル重複の排除:** 同一リール内での絵柄重複をロジックで制御。データの持ち方を工夫し、視覚的な違和感をなくすことでゲームとしての完成度を高めました。

- **サウンドによる演出:** `select.wav`（選択）、`spin.wav`（回転）、`coin.wav`（的中）の3種類の音源を実装。それぞれのゲーム進行状況（ステート）に応じて適切な効果音を呼び出す仕組みを構築し、スロット特有の期待感を演出しました。

## 2. Smart Travel（旅行支援Webアプリ）
大学の実験におけるグループワークで制作した、オープンデータを活用した観光検索システムです。

### 🎥 動作デモ

https://github.com/user-attachments/assets/b5ab0def-889c-4f5b-8aec-26328d70dc8e

### 🛠 使用技術
- **フレームワーク:** Python (Flask)
- **データベース:** SQLite3 (SQLAlchemy)
- **フロントエンド:** HTML / CSS / JavaScript

### 📑 設計・ドキュメント（担当範囲）
- **[機能解説テキスト](./web_app/docs/program_explanation.txt)**：私が中心となり、バックエンドからフロントエンドまで、**システム全体の機能詳細をドキュメント化**しました。
- **[画面遷移図](./web_app/docs/Screen%20transition%20diagram.png)**：チーム内の共通認識を作るために作成した、システムの全体設計図です。

### ✨ グループ開発での貢献
- **バックエンド担当:** CSV形式の観光データをデータベース化し、Flaskを用いた動的な検索機能を実装しました。また、AIを開発パートナーとして活用し、SQLクエリの最適化やエラーハンドリングの効率化を図りました。

- **データベース設計書（テーブル定義）**：エンティティごとの構造を定義した設計資料です。
